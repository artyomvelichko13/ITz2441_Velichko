## 🎯 Описание задачи

**Задача:** Семантическая сегментация спутниковых снимков для автоматического определения типов земной поверхности.

**Классы сегментации:**
- **Класс 0:** Фон (чёрный)
- **Класс 1:** Здания (красный)
- **Класс 2:** Дороги (серый)
- **Класс 3:** Вода (синий)
- **Класс 4:** Поля (зелёный)

**Тип задачи:** Многоклассовая семантическая сегментация (pixel-wise classification)

---

## 💡 Цель и мотивация

### Цель проекта
Разработать и обучить модель глубокого обучения для автоматической сегментации спутниковых снимков с целью классификации различных типов земной поверхности.

### Мотивация
1. **Мониторинг земель:** Автоматизация анализа спутниковых снимков для сельскохозяйственного мониторинга
2. **Городское планирование:** Определение застроенных территорий и инфраструктуры
3. **Экологический контроль:** Отслеживание водных ресурсов и изменений ландшафта
4. **Оценка последствий ЧС:** Быстрый анализ изменений после наводнений, пожаров и других катастроф

## 📊 Данные

### Источник данных

**SpaceNet SN8 Floods** - датасет спутниковых снимков для детектирования наводнений
- **Ссылка:** [SpaceNet on AWS](https://spacenet.ai/)
- **Размер:** ~7 ГБ
- **Количество изображений:** 4346
- **Формат:** GeoTIFF (.tif), PNG (.png)
- **Разрешение:** Высокое разрешение, resize до 256×256

### Обработка данных

**Проблема:** SpaceNet SN8 предоставляет векторные аннотации (GeoJSON), а не растровые маски.

**Решение:** Синтетическая генерация масок на основе реальных спутниковых снимков SpaceNet.

## 🏗️ Архитектура модели

### U-Net

**Выбрана архитектура U-Net** - стандарт для задач семантической сегментации.

#### Обоснование выбора:

1. **Эффективность:** Отличная производительность на задачах сегментации
2. **Skip connections:** Сохранение пространственной информации
3. **Простота обучения:** ~31M параметров
4. **Проверенность:** Широко используется в геопространственных задачах

#### Структура:

```
Input (3, 256, 256)
    ↓
[Encoder] → 64 → 128 → 256 → 512 channels
[Bottleneck] → 1024 channels
[Decoder] → 512 → 256 → 128 → 64 channels
    ↓
Output (5, 256, 256)
```

**Параметры модели:** 31,043,781

## 📈 Метрики качества

### IoU (Intersection over Union)

Отношение площади пересечения к площади объединения предсказанной и истинной масок.

### Dice Coefficient

Гармоническое среднее между precision и recall для пиксельной классификации.

## 🚀 Установка и запуск

### Установка

```bash
git clone https://github.com/your-username/satellite-segmentation.git
cd satellite-segmentation
pip install -r requirements.txt
```

### Подготовка данных

```bash
# Синтетические данные (быстро)
python src/create_local_data.py

# Или реальные SpaceNet (медленно)
python src/download_spacenet_windows.py
python src/add_masks_to_spacenet.py
```

### Обучение

```bash
# Быстрая демонстрация (3 эпохи)
python src/demo_pipeline.py

# Полное обучение (20 эпох)
python src/train_segmentation.py
```

### Инференс

```bash
python src/inference.py --image path/to/image.png
```
