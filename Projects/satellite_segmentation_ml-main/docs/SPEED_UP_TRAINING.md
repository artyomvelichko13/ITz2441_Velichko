# ⚡ Как ускорить обучение

## 🐌 Проблема: 30 минут на эпоху

У вас **4346 изображений** → **30 минут на эпоху** → **1.5 часа на 3 эпохи**

Это ОЧЕНЬ долго для CPU!

---

## ✂️ РЕШЕНИЕ: Уменьшить количество данных

### Способ 1: Автоматический (Рекомендуется)

```powershell
python reduce_dataset.py
```

**Что делает:**
1. Показывает текущий размер датасета
2. Предлагает выбрать целевой размер
3. Случайно выбирает нужное количество примеров
4. Удаляет остальные
5. Создаёт резервную копию

**Рекомендуемые размеры:**
- 🟢 **200 примеров** - ~6-8 минут на эпоху (быстро!)
- 🟡 **500 примеров** - ~15-20 минут на эпоху (умеренно)
- 🔴 **1000 примеров** - ~30-40 минут на эпоху (медленно)

---

### Способ 2: Ручной (быстрый)

#### Вариант A: Оставить первые N файлов

```powershell
# PowerShell скрипт
cd data\images
Get-ChildItem | Select-Object -First 200 | Move-Item -Destination ..\..\data_keep\images\
Get-ChildItem | Remove-Item

cd ..\masks
Get-ChildItem | Select-Object -First 200 | Move-Item -Destination ..\..\data_keep\masks\
Get-ChildItem | Remove-Item

cd ..\..
Move-Item data_keep\* data\
```

#### Вариант Б: Использовать файловый менеджер

1. Откройте `data\images`
2. Выделите первые 200 файлов (Shift + клик)
3. Переместите в временную папку
4. Удалите остальные файлы
5. Верните 200 файлов обратно
6. Повторите для `data\masks`

## 💡 Дополнительные способы ускорения

### 1. Уменьшить размер изображений

В `train_segmentation.py` или `demo_pipeline.py`:

```python
# Было:
IMG_SIZE = 256

# Стало:
IMG_SIZE = 128  # В 4 раза быстрее!
```

### 2. Уменьшить batch size (если мало памяти)

```python
# Было:
BATCH_SIZE = 8

# Стало:
BATCH_SIZE = 4  # Медленнее, но меньше памяти
```

### 3. Уменьшить количество эпох

```python
# Было:
NUM_EPOCHS = 20

# Стало:
NUM_EPOCHS = 3  # Для быстрого тестирования
```

### 4. Комбинация (САМЫЙ БЫСТРЫЙ вариант)

```python
IMG_SIZE = 128        # Маленькие изображения
BATCH_SIZE = 16       # Больший batch
NUM_EPOCHS = 3        # Мало эпох
# + 200 примеров

# Результат: ~2-3 минуты на всё обучение!
```
