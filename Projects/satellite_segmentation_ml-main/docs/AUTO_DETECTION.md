## 📝 Код, который это делает

В файле `train_segmentation.py` (строки 453-474):

```python
# Проверяем наличие реальных данных
data_dir = Path('data')

if (data_dir / 'images').exists() and (data_dir / 'masks').exists():
    # ✅ НАЙДЕНО! Используем реальные данные
    print("Используются реальные данные SpaceNet")
    train_dataset = SpaceNetDataset(
        image_dir=data_dir / 'images',
        mask_dir=data_dir / 'masks',
        img_size=IMG_SIZE
    )
else:
    # ❌ НЕ НАЙДЕНО! Создаём синтетические
    print("⚠️  Реальные данные не найдены.")
    train_dataset = SyntheticDataset(
        num_samples=800, 
        img_size=IMG_SIZE
    )
```

## 💡 Важные моменты

### ✅ Автоматическое обнаружение
- Программа **САМ** проверяет наличие папки `data/`
- Не нужно ничего настраивать или указывать пути
- Работает "из коробки"

### ✅ Поддерживаемые форматы
- `.png` - основной формат
- `.tif` / `.tiff` - тоже поддерживается
- `.jpg` / `.jpeg` - можно добавить (нужно модифицировать код)

### ✅ Именование файлов
- Имена файлов могут быть любыми
- Главное - чтобы были в правильных папках
- Маски могут иметь такие же имена, как изображения

### ✅ Количество файлов
- Минимум: 10-20 файлов (для тестирования)
- Оптимально: 500-2000 файлов
- Максимум: программа обработает любое количество

## ❓ FAQ

**Q: Нужно ли что-то менять в коде после скачивания данных?**  
A: **НЕТ!** Программа автоматически обнаружит данные.

**Q: Как узнать, какие данные используются?**  
A: Запустите `python check_data.py` или смотрите в консоль при запуске train_segmentation.py

**Q: Можно ли использовать часть данных SpaceNet?**  
A: Да! Просто скопируйте нужные файлы в `data/images` и `data/masks`

**Q: Что если имена файлов изображений и масок не совпадают?**  
A: Программа всё равно загрузит их, но могут быть несоответствия. Лучше, чтобы имена совпадали.

**Q: Можно ли смешать SpaceNet и синтетические данные?**  
A: Да! Просто положите все файлы в `data/images` и `data/masks`
